## Software

Download [Raspbian](https://www.raspberrypi.org/downloads/raspbian/)

Flash with [Etcher](https://www.balena.io/etcher/)

Enable SSH with 

```sh
$ touch /Volumes/boot/ssh
```

Eject the disk volume

```sh
$ sudo umount /Volumes/boot/ssh
```
